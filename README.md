> Play lets you connect with people who share your interests.


## Developing


### Built With

* React.js
* Ant-design
* OAuth2.0
* Socket.io

### Prerequisites
* **Node** should be installed on your machine.


### Setting up Dev


```shell
git clone https://github.com/Kushagrabainsla/play.git
cd play
```

```shell
npm install
npm start
```
